Usage
--------------------------------------------------------------------------------
此工具为在线收集如下信息：
	获取虚拟机信息并写入数据库
	获取ESX主机信息并定入数据库，此信息为特定信息，主要为ESX主机CPU、内存分配率
	
Quick Start
--------------------------------------------------------------------------------
	./control.sh

Requirements
--------------------------------------------------------------------------------
    python3
    pyVmoni==6.0.0.2016.4
    openpyxl==2.3.0b1
    xlrd==1.0.0
    sqlite3
